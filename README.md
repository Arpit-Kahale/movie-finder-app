# 🎬 CineSearch - Movie Search App

A responsive Movie Search Web Application built using React and Tailwind CSS. This app uses the OMDb API to search movies, filter results, and view detailed movie information.

---

## 🚀 Features

- 🔍 Search movies by name
- 🎭 Filter by type (Movie / Series)
- 📃 View detailed movie information
- 🖼 Poster fallback for missing images
- ⚡ Fast and responsive UI
- 📱 Mobile-friendly design
- 🔝 Sticky Navbar and Search Bar

---

## 🛠 Tech Stack

- React.js
- Tailwind CSS
- Axios
- OMDb API
- React Router DOM
- Vite


## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/Arpit-Kahale/movie-finder-app.git
````

2. Navigate to project folder:

```bash
cd cine-search-app
```

3. Install dependencies:

```bash
npm install
```

4. Run the app:

```bash
npm run dev
```
